# GitHub Expo

AI-powered GitHub Expo — bounty scraper & money maker that finds highest paying open programs on HackerOne, Bugcrowd, Intigriti, etc.

## Description
This scraper finds the highest-paying open bug bounty programs from HackerOne, Bugcrowd, and Intigriti, ranks them, and saves the results to `bounties.json`.

## Usage
1. Install dependencies: `pip install -r requirements.txt`
2. Run the scraper: `python main.py`
